# Railway UI操作ガイド

## ステップ1: New Projectをクリック
✅ 完了済み

## ステップ2: GitHub Repositoryを選択
✅ 完了済み（画像で選択されている）

## ステップ3: リポジトリを選択

「GitHub Repository」をクリックすると、次の画面に進みます：

1. **リポジトリ検索画面が表示されます**
2. 検索ボックスに `kitayama-ai/portfolio` と入力
3. リポジトリが表示されたら、クリックして選択

## ステップ4: デプロイ設定

リポジトリを選択すると、以下の設定画面が表示されます：

### 設定項目

1. **Service Name**（自動入力される場合あり）
   - 例: `portfolio` または `zoom-recorder-web`

2. **Root Directory**
   - **重要**: `zoom-recorder-web/backend` に設定
   - これがないと、デプロイが失敗します

3. **Start Command**
   - `python3 main.py`（自動検出される場合あり）

4. **Environment Variables**
   - 後で設定可能（デプロイ後でも設定できます）

### デプロイボタン

設定画面の下部に以下のボタンが表示されます：
- **「Deploy」** または **「Deploy Now」** ボタン

このボタンをクリックするとデプロイが開始されます。

## トラブルシューティング

### リポジトリが表示されない場合

1. GitHubアカウントが正しく連携されているか確認
2. 「Connect GitHub」をクリックして再認証
3. リポジトリ名を正確に入力（`kitayama-ai/portfolio`）

### デプロイボタンが表示されない場合

1. リポジトリが正しく選択されているか確認
2. ページをリロード
3. ブラウザのキャッシュをクリア

### 次の画面に進めない場合

1. 「GitHub Repository」をクリック
2. リポジトリ検索画面が表示されるはずです
3. 表示されない場合は、ページをリロード
